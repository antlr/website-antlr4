ANTLR Discussions

Please do not start discussions at stackoverflow. They have asked us to steer discussions (i.e., non-questions/answers) away from Stackoverflow; we have a discussion forum at GitHub specifically for that:

https://github.com/antlr/antlr4/discussions

We can discuss ANTLR project features, direction, and generally argue about whatever we want in GitHub discussion area.
